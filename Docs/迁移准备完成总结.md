## 🚀 下一步操作

### 当前电脑操作
```bash
# 1. 推送到远程仓库
git push

# 2. 运行迁移准备脚本（可选）
./migration-prepare.sh
```

### 新电脑操作
```bash
# 1. 克隆项目
git clone [你的仓库地址] MyGarden
cd MyGarden

# 2. 安装依赖
npm install

# 3. 安装云函数依赖
cd cloudfunctions/login && npm install
cd ../user && npm install
cd ../garden && npm install
cd ../carbon && npm install
cd ../..

# 4. 启动开发
npm run dev:weapp
```
