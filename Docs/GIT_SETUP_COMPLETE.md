## 🏗️ 分支结构

```
main          - 生产环境代码 (保护分支)
develop       - 开发集成分支
feature/*     - 功能开发分支
hotfix/*      - 紧急修复分支
release/*     - 版本发布分支
```

## 🔄 开发工作流

1. **功能开发** → 从develop创建feature分支
2. **代码审查** → PR到develop分支，至少2人审查
3. **集成测试** → develop分支自动化测试
4. **版本发布** → 从develop创建release分支
5. **生产部署** → release合并到main分支
