# 气候餐厅管理后台 - 部署快速指南

## 🎯 推荐部署方式

**腾讯云静态网站托管** ⭐⭐⭐⭐⭐

### 为什么推荐？

- ✅ 与现有腾讯云开发环境完美集成
- ✅ 自动 CDN 加速，访问速度快
- ✅ 成本低，适合中小型项目
- ✅ 部署简单，一条命令完成
- ✅ 支持自定义域名和 HTTPS

---

## 🚀 快速部署（3 步完成）

### 方法一：使用部署脚本（推荐）

```bash
cd admin-web
npm run deploy
```

### 方法二：手动部署

```bash
# 1. 构建
cd admin-web
npm install
npm run build

# 2. 部署
tcb hosting:deploy dist -e my-garden-app-env-4e0h762923be2f
```

---

## 📋 部署前准备

1. **安装 CloudBase CLI**

   ```bash
   npm install -g @cloudbase/cli
   ```

2. **登录腾讯云**

   ```bash
   tcb login
   ```

3. **检查登录状态**
   ```bash
   tcb login:check
   ```

## 🐛 常见问题

### SPA 路由 404 错误

在腾讯云控制台配置"错误页面"为 `index.html`

### 静态资源 404

检查 `vite.config.ts` 中的 `base` 配置

### 云函数调用失败

检查云开发环境 ID 和匿名登录状态

**更多问题请查看：** `DEPLOY.md` 中的"常见问题"章节
