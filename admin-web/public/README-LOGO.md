# Logo 文件说明

## 需要放置的 Logo 文件

请将以下 Logo 文件放置在此目录（`admin-web/public/`）：

### 1. 完整 Logo

- **文件名**: `logo.svg` 或 `logo.png` 或 `logo.jpg` 或 `logo.webp`
- **用途**: 侧边栏展开时显示
- **格式支持**: SVG（推荐）、PNG、JPG、WebP
- **内容**: 包含"谷秀"中文字符和"GOOD SHOW CONSULTING"英文文本

### 2. 简化 Logo（可选）

- **文件名**: `logo-icon.svg` 或 `logo-icon.png` 或 `logo-icon.jpg` 或 `logo-icon.webp`
- **用途**: 侧边栏折叠时显示
- **格式支持**: SVG（推荐）、PNG、JPG、WebP
- **内容**: 简化版 Logo 或图标

## Logo 规格建议

### 完整 Logo

- **宽度**: 建议 160-200px
- **高度**: 建议 40-50px
- **格式优先级**:
  1. **SVG**（推荐）- 矢量图，任意缩放不失真，文件小
  2. **PNG** - 支持透明背景，适合复杂 Logo
  3. **WebP** - 现代格式，压缩率高
  4. **JPG** - 通用格式，但不支持透明背景
- **内容**: 包含"谷秀"中文字符、"GOOD SHOW CONSULTING"英文文本和红色印章元素

### 简化 Logo

- **宽度**: 建议 32-40px
- **高度**: 建议 32-40px
- **格式优先级**: 同上
- **内容**: 简化版图标或单个字符

## 格式选择建议

### 推荐使用 SVG

- ✅ 矢量图，任意缩放不失真
- ✅ 文件体积小
- ✅ 支持透明背景
- ✅ 适合 Logo 这种简单图形

### 如果使用 PNG

- ✅ 支持透明背景
- ✅ 兼容性好
- ⚠️ 需要准备不同尺寸（1x、2x）以适配高分辨率屏幕
- ⚠️ 文件体积较大

### 如果使用 JPG

- ✅ 兼容性最好
- ⚠️ 不支持透明背景（需要白色背景）
- ⚠️ 文件体积较大

## 自动降级机制

组件会自动尝试加载不同格式：

1. 首先尝试 SVG 格式（`logo.svg`）
2. 如果 SVG 不存在，自动尝试 PNG 格式（`logo.png`）
3. 如果都不存在，显示文字"谷秀"作为备用

## 使用说明

### 文件命名规则

组件会按以下优先级查找 Logo 文件：

**完整 Logo（展开状态）**:

1. `logo.svg`（优先）
2. `logo.png`
3. `logo.jpg`
4. `logo.webp`

**简化 Logo（折叠状态）**:

1. `logo-icon.svg`（优先）
2. `logo-icon.png`
3. `logo-icon.jpg`
4. `logo-icon.webp`

### 自动降级机制

- 如果 SVG 文件不存在，会自动尝试 PNG
- 如果所有图片格式都不存在，会显示文字"谷秀"作为备用
- 无需修改代码，只需放置文件即可

### 放置 Logo 文件

**最简单的方式**:

1. 将 Logo 图片重命名为 `logo.png`（或 `logo.svg`）
2. 放置到 `admin-web/public/` 目录
3. 刷新页面即可看到效果

**推荐方式**:

- 如果有 SVG 格式，使用 `logo.svg`（最佳）
- 如果有 PNG 格式，使用 `logo.png`（也很好）
- 如果有 JPG 格式，使用 `logo.jpg`（可用，但不支持透明背景）
