# 图片资源目录

此目录用于存放小程序所需的图片资源。

## 需要的图片文件：
- `home.png` - 首页图标
- `home-active.png` - 首页激活图标
- `chat.png` - 聊天图标
- `chat-active.png` - 聊天激活图标
- `analyze.png` - 分析图标
- `analyze-active.png` - 分析激活图标
- `logo.png` - 应用Logo

## 使用说明：
图片文件需要手动添加到此目录中。在微信开发者工具中，可以通过以下方式添加：
1. 右键点击images文件夹
2. 选择"上传文件"
3. 选择对应的图片文件

## 图片规格建议：
- 图标尺寸：60x60像素
- Logo尺寸：120x120像素
- 格式：PNG（支持透明背景）
